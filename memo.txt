New Structure

gemini_translator_v4/
├── main.py                  # Entry point (recommended)
├── translator.py            # Backward compatible (just imports main)
├── config.py                # Configuration (unchanged)
│
└── src/
    ├── constants.py         # VERSION, LANGUAGES, PROVIDERS_LIST, etc.
    ├── app.py               # TranslatorApp class (~700 lines)
    │
    ├── core/
    │   ├── clipboard.py     # ClipboardManager
    │   ├── api_manager.py   # AIAPIManager
    │   ├── translation.py   # TranslationService
    │   └── hotkey.py        # HotkeyManager
    │
    ├── ui/
    │   ├── settings.py      # SettingsWindow (~960 lines)
    │   └── dialogs.py       # APIErrorDialog
    │
    └── utils/
        ├── logging_setup.py # setup_logging()
        ├── single_instance.py # is_already_running()
        └── updates.py       # check_for_updates(), download_and_install_update()
How to Use
Task	Command/File
Run app (new way)	python main.py
Run app (old way)	python translator.py (works same as before)
Build exe	PyInstaller will work the same
When Editing Code
Want to change...	Edit this file
Constants (VERSION, LANGUAGES)	src/constants.py
Clipboard logic	src/core/clipboard.py
API providers	src/core/api_manager.py
Translation prompts	src/core/translation.py
Hotkey registration	src/core/hotkey.py
Settings UI	src/ui/settings.py
Error dialogs	src/ui/dialogs.py
Main app (tray, tooltip, popup)	src/app.py
Update checker	src/utils/updates.py
